## 📋 نظرة عامة

**عُدّة القارئ للملفات الرقمية** هو تطبيق سطح مكتب متكامل وقوي يوفر مجموعة شاملة من الأدوات المتقدمة لمعالجة وإدارة ملفات PDF. يتميز التطبيق بواجهة مستخدم عصرية تدعم اللغتين العربية والإنجليزية مع تجربة مستخدم محسنة ومميزات احترافية.

**PDF Tools** is a comprehensive desktop application providing advanced tools for PDF manipulation and management. Features a modern bilingual interface (Arabic/English) with professional capabilities and enhanced user experience.

## ✨ المميزات الرئيسية (15 أداة متخصصة)

### 📖 **1. عارض PDF المتقدم** (Advanced PDF Viewer)
عارض PDF احترافي مع أدوات تعليق متقدمة، دعم الإشارات المرجعية، التكبير والتصغير، البحث في النصوص، وحفظ موضع القراءة تلقائياً.

### 📚 **2. مكتبة الكتب الشخصية** (Personal Book Library)
نظام متكامل لإدارة مكتبتك الرقمية مع تتبع تقدم القراءة، إحصائيات مفصلة، تصنيف الكتب، إضافة تقييمات، واستئناف القراءة من حيث توقفت.

### ⚡ **3. قياس سرعة القراءة** (Reading Speed Tracker)
أداة ذكية لقياس سرعة قراءتك مع تمارين تدريبية متقدمة. تتبع تقدمك، شاهد تحسن أدائك عبر رسوم بيانية تفاعلية.

### 📝 **4. استخراج النصوص** (Text Extraction)
استخراج النصوص بدقة عالية مع دعم تصدير بصيغ متعددة (TXT، JSON، DOCX) مع الحفاظ على التنسيق.

### 🔖 **5. إدارة الإشارات المرجعية** (Bookmark Manager)
أداة قوية لإنشاء وتحرير وتنظيم الإشارات المرجعية. أضف فصولاً جديدة، عدّل العناوين، غيّر الترتيب الهرمي.

### 📑 **6. استخراج الفهارس** (Bookmark Extractor)
استخراج الإشارات المرجعية وحفظها بصيغ متعددة (نصية أو Excel) مع تفاصيل كاملة عن أرقام الصفحات والمستويات الهرمية.

### 📂 **7. تقسيم PDF حسب الفهارس** (Divide PDF by Bookmarks)
تقسيم الكتب الكبيرة بذكاء! حوّل كل فصل إلى ملف مستقل مع الاحتفاظ بالتنسيق الأصلي.

### 📊 **8. تحليل أوزان الفصول** (Chapter Weight Analysis)
أداة تحليلية متقدمة لحساب أوزان الفصول وإنشاء خطط قراءة ذكية. صدّر التحليلات إلى Excel.

### 📄 **9. عمليات الصفحات** (Page Operations)
تحكم كامل في صفحاتك! استخرج صفحات محددة، احذف صفحات غير مرغوبة، أدرج صفحات فارغة، أو ادمج ملفات متعددة.

### 🛡️ **10. العلامات المائية** (Watermark Tools)
أضف علامات مائية نصية أو صور لحماية ملكيتك الفكرية. تحكم كامل في الموضع، الشفافية، الحجم، والدوران.

### 🖼️ **11. استخراج الصور** (Image Extraction)
استخراج جميع الصور بجودة عالية! احفظ الصور بصيغ متعددة (PNG، JPEG) مع الحفاظ على الدقة الأصلية.

### 📋 **12. دمج ملفات PDF** (PDF Merger)
ادمج عدة ملفات PDF مع الحفاظ على الجودة. أعد ترتيب الملفات قبل الدمج، عاين النتيجة النهائية.

### 🗜️ **13. ضغط الملفات** (PDF Compression)
قلّل الحجم بذكاء! اضغط ملفاتك مع الحفاظ على جودة مقبولة. خيارات ضغط متعددة (عالي، متوسط، منخفض).

### ✂️ **14. تحرير وتنظيم الصفحات** (Edit & Organize Pages)
عدّل صفحاتك باحترافية! دوّر الصفحات، اقصص الهوامش، أعد ترتيب الصفحات، وغيّر الأحجام.

### 🔓 **15. إزالة الحماية** (Remove Security)
أزل كلمات المرور والقيود الأمنية من ملفاتك (التي تملك حقوق الوصول إليها).

## 📥 التحميل والتثبيت

### التحميل المباشر (موصى به):
1. **[تحميل أحدث إصدار](https://github.com/zemam-injaz/PDF-Tools/releases/latest)**
2. قم بتحميل ملف `PDF-Tools-v2.0.0-Setup.exe` (Windows Installer)
3. شغل المثبت واتبع التعليمات
4. ابدأ التطبيق من قائمة ابدأ أو سطح المكتب

### التحميل المحمول (Portable):
1. قم بتحميل ملف `PDF-Tools-v2.0.0-Portable.zip`
2. استخرج الملفات إلى مجلد من اختيارك
3. شغل `PDF_Tools_Pro.exe`

### التثبيت من المصدر (للمطورين):
```bash
# استنساخ المستودع
git clone https://github.com/zemam-injaz/PDF-Tools.git
cd PDF-Tools

# إنشاء بيئة افتراضية
python -m venv venv
source venv/Scripts/activate  # Windows
# source venv/bin/activate     # Linux/Mac

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python src/pdf_tools_comprehensive.py
```

## 🛠️ البناء من المصدر

### إنشاء ملف تنفيذي (Executable):
```bash
# استخدم السكريبت المُعد مسبقاً
cd build_config
build.bat
```

سيتم إنشاء الملف التنفيذي في: `build_config/dist/PDF_Tools_Pro.exe`

### إنشاء مثبت (Installer):
بعد إنشاء الملف التنفيذي، استخدم Inno Setup مع الملف:
```
build_config/pdf_tools_pro_installer.iss
```
